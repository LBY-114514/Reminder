# 挑战杯提醒

这是一个电脑本地使用的小工具，用来记录挑战杯相关的问题、任务和提醒时间。

启动后自动缩到系统托盘（右下角通知区域），不弹黑窗口。在网页界面里可以添加事项、修改事项、标记完成、删除事项，并设置提醒时间。到时间后，电脑会弹出提醒，网页右上角也会显示站内提醒。

## 适合谁用

- 想记录挑战杯准备过程中遇到的问题
- 想给自己设置复盘、修改、提交材料等提醒
- 想要一个简单、本地、不用注册账号的提醒工具

## 怎么使用

### 1. 下载

下载 Release 中的 `Reminder.zip`，解压得到 `挑战杯提醒.exe`。

### 2. 双击运行

双击 `挑战杯提醒.exe`，程序会在系统托盘（右下角通知区域）出现一个图标，同时自动打开浏览器页面。

如果浏览器没有自动打开，可以右键托盘图标 → **打开界面**，或手动访问：

```text
http://localhost:8787
```

### 3. 添加提醒

在页面里填写：

- 标题
- 内容
- 提醒时间

然后点击添加按钮即可。

提醒时间到了之后，电脑会弹出提醒窗口，网页右上角也会弹出站内提醒卡片。

程序会定时检查提醒，通常会在设定时间后的几秒内弹出。

### 4. 退出程序

右键托盘图标 → **退出** 即可完全关闭程序。

关闭后提醒不会继续弹出，需要保持程序运行才能收到提醒。

## 提醒音效

页面里有一个提醒音效设置。你可以选择一个自己的 mp3 文件作为提醒音效。

选择后，程序会把这个 mp3 复制到当前数据保存文件夹里，并命名为 `reminder-sound.mp3`。

注意：

- 只支持 `.mp3` 文件。
- 选择后原来的 mp3 可以移动或删除，因为程序已经复制了一份。
- 如果你更换数据保存位置，程序会尽量把音效文件一起复制到新的数据文件夹。
- 如果音效文件被手动删除，提醒仍会弹窗，只是不会播放声音。

## 开机自动启动

页面里有一个开机自动启动设置。

如果你希望电脑开机后自动运行这个工具，点击 **开启开机启动**。如果之后不想开机自动运行，点击 **关闭开机启动**。

注意：开启开机启动后，不要随便移动 `挑战杯提醒.exe` 的位置，否则开机启动可能会失效。如果移动了 exe 文件，重新打开程序，再点一次开启开机启动即可。

## 设置数据保存位置

程序会把你记录的内容保存成一个 JSON 文件。

如果你想自己指定保存位置，可以在页面路径输入框中粘贴文件夹地址（比如 `D:\ReminderData`），然后点击 **保存到这个文件夹**。

### 注意：路径不要带引号

从 Windows 地址栏复制路径时可能会带上引号（如 `"D:\ReminderData"`），需要把前后的引号删掉，否则程序可能无法正确识别。

## 数据会保存在哪里

- 如果没有手动设置保存位置，程序使用默认位置。
- 如果手动设置了保存文件夹，数据保存在你选择的文件夹里。下次打开程序会继续使用上次的设置。
- 如果选择了提醒音效，音效文件也保存在同一个文件夹里。

## 常见问题

### 运行后没有自动打开网页怎么办？

右键系统托盘图标 → **打开界面**，或手动在浏览器访问 `http://localhost:8787`。

### 关闭程序后还能提醒吗？

不能。提醒功能需要程序正在运行。保持程序在后台运行（托盘图标在右下角即可），不要退出。

### 怎么知道程序还在运行？

看右下角系统托盘有没有程序图标。如果没有，说明程序已退出，重新双击 exe 启动即可。

### 提醒为什么会晚几秒？

程序每 5 秒检查一次提醒，所以提醒可能比设置时间晚几秒，这是正常现象。

### Windows 提示阻止 exe 怎么办？

如果 Windows 提示"智能应用控制已阻止可能不安全的应用"，通常是因为这个 exe 是本地打包生成、没有代码签名。可以在 Windows 安全中心里按自己的判断允许该程序。

### 需要安装 Python 或 Conda 吗？

不需要。直接双击 `挑战杯提醒.exe` 运行即可。

### 数据会上传到网上吗？

不会。这是本地工具，数据保存在你自己的电脑里。
