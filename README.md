# 挑战杯提醒

这是一个电脑本地使用的小工具，用来记录挑战杯相关的问题、任务和提醒时间。

打开后会自动进入一个网页界面，你可以在里面添加事项、修改事项、标记完成、删除事项，并设置提醒时间。到时间后，电脑会弹出提醒，网页右上角也会显示站内提醒。

## 适合谁用

- 想记录挑战杯准备过程中遇到的问题
- 想给自己设置复盘、修改、提交材料等提醒
- 想要一个简单、本地、不用注册账号的提醒工具

## 怎么使用

### 1. 下载 exe 文件

下载这个文件：
打开旁边的release
```text
Reminder.zip
```

### 2. 双击运行
解压后
双击 `挑战杯提醒.exe`。

程序启动后会自动打开浏览器页面。如果没有自动打开，可以看程序窗口里显示的本地地址，一般类似：

```text
http://localhost:8787
```

把这个地址复制到浏览器里打开即可。

### 3. 添加提醒

在页面里填写：

- 标题
- 内容
- 提醒时间

然后点击添加按钮即可。

提醒时间到了之后，电脑会弹出提醒窗口，网页右上角也会弹出站内提醒卡片。

程序会定时检查提醒，通常会在设定时间后的几秒内弹出，不一定完全卡在同一秒。

## 提醒音效

页面里有一个：

```text
提醒音效
```

你可以选择一个自己的 mp3 文件作为提醒音效。

选择后，程序会把这个 mp3 复制到当前数据保存文件夹里，并命名为：

```text
reminder-sound.mp3
```

以后提醒时，如果音效开关是开启状态，弹窗出现时会同时播放这个音效。

注意：

- 只支持 `.mp3` 文件。
- 选择后原来的 mp3 可以移动或删除，因为程序已经复制了一份。
- 如果你更换数据保存位置，程序会尽量把音效文件一起复制到新的数据文件夹。
- 如果音效文件被手动删除，提醒仍会弹窗，只是不会播放声音。

## 开机自动启动

页面里有一个：

```text
开机自动启动
```

如果你希望电脑开机后自动运行这个工具，可以点击：

```text
开启开机启动
```

如果之后不想开机自动运行，可以点击：

```text
关闭开机启动
```

注意：开启开机启动后，不要随便移动 `挑战杯提醒.exe` 的位置，否则开机启动可能会失效。

如果移动了 exe 文件，重新打开程序，再点一次开启开机启动即可。

## 设置数据保存位置

程序会把你记录的内容保存成一个 JSON 文件。

如果你想自己指定保存位置，可以在页面里的路径输入框中粘贴文件夹地址，例如：

```text
D:\ReminderData
```

然后点击：

```text
保存到这个文件夹
```

程序会自动在这个文件夹里保存数据文件。

### 注意：路径不要带引号

从 Windows 地址栏复制路径时，有些情况下可能会带上引号，例如：

```text
"D:\ReminderData"
```

这种情况需要把前后的引号删掉，改成：

```text
D:\ReminderData
```

否则程序可能无法正确识别路径。

## 数据会保存在哪里

如果你没有手动设置保存位置，程序会使用默认位置保存数据。

如果你手动设置了保存文件夹，数据会保存在你选择的文件夹里。

以后再次打开程序，会继续使用上一次设置的保存位置。

如果你选择了提醒音效，音效文件也会保存在同一个文件夹里。

## 常见问题

### 运行后没有自动打开网页怎么办？

看一下程序窗口里显示的地址，把类似下面的地址复制到浏览器打开：

```text
http://localhost:8787
```

### 关闭程序后还能提醒吗？

不能。提醒功能需要程序正在运行。

如果你希望它提醒你，记得保持 `挑战杯提醒.exe` 不要关闭。

### 怎么完全退出程序？

只关闭浏览器页面不一定会结束程序。

如果想完全退出，请回到运行 `挑战杯提醒.exe` 的终端窗口，按：

```text
Ctrl + C
```

这样才能结束程序进程。

### 误关网页或终端怎么办？

如果只是关闭了浏览器网页，程序通常还在终端里运行，提醒仍然会继续弹出。可以在终端里找到本地地址，例如：

```text
http://localhost:8787
```

再复制到浏览器打开即可。

如果误关了终端窗口，程序一般也会一起关闭，提醒不会继续弹出。重新双击 `挑战杯提醒.exe` 启动即可。

如果你不确定程序是否还在后台运行，可以打开任务管理器，查找并结束：

```text
挑战杯提醒.exe
```

### 提醒为什么会晚几秒？

程序为了减少占用，不是每一毫秒都检查提醒，而是定时检查一次。

当前版本大约每 5 秒检查一次，所以提醒可能会比设置时间晚几秒，这是正常现象。

### Windows 提示阻止 exe 怎么办？

如果 Windows 提示“智能应用控制已阻止可能不安全的应用”，通常是因为这个 exe 是本地打包生成、没有代码签名，不一定代表程序有问题。

如果当前版本可以正常运行，建议不要频繁改动 exe。遇到阻止时，可以改用 `start.bat` 启动源码版本，或在 Windows 安全中心里按自己的判断允许该程序。

### 需要安装 Python 或 Conda 吗？

不需要。

如果你下载的是 `挑战杯提醒.exe`，直接双击运行即可。

### 数据会上传到网上吗？

不会。

这是本地工具，数据保存在你自己的电脑里。
